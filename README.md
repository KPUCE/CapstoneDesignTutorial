# CapstoneDesignTutorial: 한국공대 캡스톤디자인 가이드

## 오픈 캡스톤디자인 주제

### 스마트폰을 이용한 영상 제작 서비스 앱
  * 여러대의 스마트 폰으로 촬영한 동영상을 하나로 편집하여 완성도 높은 동영상 제작
  * 예를 들면 한국공대 체육대회 축구중계, 배구중계, 농구중계 등

### 스마트디바이스를 활용한 서비스
  * 스마트글래스, 스마트워치, 스마트스피커, 스마트 링, 스마트태그 등 스마트 디바이스를 활용한 서비스 개발
